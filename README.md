**StepStyle**
PHP自作
製作者:久本　健斗
**概要**
靴のECサイトを作成しました。
管理者と一般ユーザーに分け、それぞれでログインできるようにしています。
**使い方**
一般ユーザー
商品の閲覧、お気に入り登録、カート追加、レビュー投稿、ログイン、アカウント編集、パスワードリセット及びクレジット支払いが可能になっています。
クレジット支払いについては、Stripe.jsを使用しております。

管理者
一般ユーザーの操作に加え、商品編集、商品追加、商品削除が可能になっています。
